# templates/

内容模板库 — 为创作类 Skills 提供结构指南和风格参考。

## 成员清单

### 母稿模板（/母稿 skill 使用）
`母稿-猎奇型.md`: 猎奇型结构指南（钩子→铺垫→核心→转折→互动），适合反常识冲击和数据震撼
`母稿-教程型.md`: 教程型结构指南（痛点→方案→步骤→效果→互动），适合手把手教学和工具分享
`母稿-互动型.md`: 互动型结构指南（话题→背景→选项→选择→互动），适合引发讨论和UGC

### 洗稿模板（/洗稿 skill 使用，from CAT5_Brain）
`洗稿-个人故事型.md`: 风格A — 以第一人称叙事重构，强代入感
`洗稿-对话场景型.md`: 风格B — 构建虚拟对话场景，信息自然嵌入
`洗稿-情感共鸣型.md`: 风格D — 情感驱动叙事，共鸣优先
`洗稿-优化版.md`: 风格D2 — D版改良，更强的反AI检测能力
`洗稿-口语重写版.md`: 风格E — 极致口语化，碎碎念风格，AI检测率最低

### 技术文章模板（/linuxdo skill 参考，from CAT5_Brain）
`技术文章-mentor.md`: V1 Mentor风格技术文（含代码块保护策略）
`技术文章-mentor-v2.md`: V2 反AI检测优化版（打破对仗+口语碎碎念）

### 管理模板
`周复盘.md`: 周复盘报告模板（产出统计/数据表现/渠道评估/类型复盘/调整方向/指标追踪）

### 参考案例（只读，选题灵感用）
`参考案例/`: 小红书参考案例 + 内容输入案例，提供风格和结构参考

## 使用方式

/母稿 → 根据内容类型读取对应母稿模板
/洗稿 → 根据风格选择读取对应洗稿模板 + creator.yaml anti_detection 策略
/linuxdo → 可参考技术文章模板
/周复盘 → 读取周复盘模板并自动填充产出统计数据
