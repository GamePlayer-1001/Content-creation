# templates/
> L2 | 父级: /CLAUDE.md

内容模板库 — 为 /母稿 skill 提供结构指南，为 /周复盘 提供报告模板。

## 成员清单

`母稿-猎奇型.md`: 猎奇型母稿结构指南（钩子→铺垫→核心→转折→互动），适合反常识冲击和数据震撼
`母稿-教程型.md`: 教程型母稿结构指南（痛点→方案→步骤→效果→互动），适合手把手教学和工具分享
`母稿-互动型.md`: 互动型母稿结构指南（话题→背景→选项→选择→互动），适合引发讨论和UGC
`周复盘.md`: 周复盘报告模板（产出统计/数据表现/渠道评估/类型复盘/调整方向/指标追踪）

## 使用方式

/母稿 skill 根据用户选择的内容类型读取对应模板作为结构参考。
/周复盘 skill 读取周复盘模板并自动填充产出统计数据。
