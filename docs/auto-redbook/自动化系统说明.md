## 🎯 系统功能

在原有的渲染和发布功能基础上，新增：
- ✅ **自动内容生成** - Claude 智能生成命理相关内容

- ✅ **定时调度系统** - 每天自动发布 3 篇内容
- ✅ **热点抓取** - 自动获取小红书热点话题
- ✅ **Playwright 自动发布** - 完整浏览器自动化
- ✅ **软性推广** - 自然推广网站，无明显广告痕迹

## ⚙️ 系统配置

### 发布时间表

**每天 3 次发布**:
- **早间** 08:00 - 轻松正能量风格（200-400字）
  - 主题：今日运势、塔罗一抽、幸运小贴士
  - 推荐主题：mint, sunset, xiaohongshu

- **午间** 12:30 - 专业干货风格（500-800字）
  - 主题：塔罗教学、命理科普、风水布局
  - 推荐主题：elegant, ocean, purple

- **晚间** 19:30 - 温暖疗愈风格（600-900字）
  - 主题：梦境解析、情感占卜、心灵成长
  - 推荐主题：dark, sunset, purple

### 推广策略

**4种软性推广风格**:
1. **无推广**（10%） - 纯内容，建立信任
2. **自然提及**（40%） - "我平时在 destinyteller.com 上查看..."
3. **资源分享**（30%） - "系统学习可以去 destinyteller.com 看看"
4. **直接推荐**（20%） - "专业塔罗师都在用 destinyteller.com"

**禁用词汇**: 广告、推广、必买、点击、关注我、私信、优惠、折扣、限时

## 🔄 完整自动化流程

### 每日工作流

```
06:00 → 抓取小红书热点话题
    ↓
07:00 → 生成早间内容 (Claude) → 渲染图片 (Playwright)
    ↓
08:00 → 自动发布到小红书 (Playwright) → 归档内容
    ↓
11:00 → 刷新热点数据
    ↓
11:30 → 生成午间内容 → 渲染图片
    ↓
12:30 → 自动发布 → 归档
    ↓
17:00 → 刷新热点数据
    ↓
17:30 → 生成晚间内容 → 渲染图片
    ↓
19:30 → 自动发布 → 归档
```

### 内容生成流程

```
1. 加载素材库
   ├─ 从 materials/topics/ 选择话题
   ├─ 从 materials/hashtags/ 选择标签
   └─ 结合当日热点数据

2. Claude 生成内容
   ├─ 按时段风格生成 Markdown
   ├─ 包含 YAML frontmatter（emoji、标题、副标题）
   └─ 自然融入软性推广

3. 渲染图片
   ├─ 使用 render_xhs_v2.js
   ├─ 按时段选择主题
   └─ 生成封面 + 多张卡片

4. 自动发布
   ├─ Playwright 打开浏览器
   ├─ 加载保存的 Cookie
   ├─ 上传图片、填写标题和描述
   ├─ 添加标签
   └─ 点击发布

5. 归档管理
   ├─ 保存发布结果（笔记 URL）
   ├─ 移动到 published/ 目录
   └─ 记录发布日志
```

## 🔧 故障排查

### Cookie 过期
```bash
# 重新测试发布，手动登录保存 Cookie
python test_publish_manual.py
```

### 内容生成失败
```bash
# 检查素材库是否存在
ls materials/topics/trending_topics.json
ls materials/hashtags/primary_tags.json

# 手动运行生成器
python automation/generators/claude_content_generator.py
```

### 图片渲染失败
```bash
# 检查 Node.js 依赖
npm install

# 检查 Playwright
playwright install chromium
```

### 自动发布失败
```bash
# 查看发布日志
cat logs/publishing/latest.log

# 检查 Cookie 文件
ls automation/config/xhs_cookies.json

# 手动重试发布
python test_publish_manual.py
```
